# AI Resume Screening System

## Problem Addressed
Manual resume screening is time-consuming for recruiters and job seekers often don't know how well their resumes match job requirements.

## Features
- Resume PDF Analysis
- Skill Extraction
- Job Match Score
- ATS Recommendations
- Interview Question Suggestions

## Technologies Used
- Python
- Flask
- HTML/CSS
- PyPDF2
